Descrição do Projeto
Este projeto implementa uma calculadora de formas geométricas utilizando Python. Ele permite calcular a área de círculos, quadrados e triângulos, bem como o perímetro de círculos, quadrados. Para triângulos, o programa exibe uma mensagem, já que o perímetro não pode ser calculado apenas com base e altura.

Como Usar o Programa
Execução do Programa:

Certifique-se de ter Python instalado em seu sistema.
Execute o arquivo calculadora_formas_geometricas.py.
Escolhendo a Forma Geométrica:

O programa solicitará que você escolha uma forma geométrica: círculo, quadrado ou triângulo.
Fornecendo as Dimensões:

Dependendo da forma escolhida, o programa solicitará as dimensões necessárias:
Para o círculo, você precisará fornecer o raio.
Para o quadrado, você precisará fornecer o lado.
Para o triângulo, você precisará fornecer a base e a altura.
Visualizando os Resultados:

O programa calculará a área e, quando aplicável, o perímetro com base nas dimensões fornecidas.
Os resultados serão exibidos no console.
Estrutura do Código
O código está organizado em três classes:

Classe Circulo:

Métodos: calcular_area(), calcular_perimetro()
Classe Quadrado:

Métodos: calcular_area(), calcular_perimetro()
Classe Triangulo:

Métodos: calcular_area(), calcular_perimetro()
Função Main:

Solicita a forma geométrica desejada e suas dimensões.
Cria objetos das classes correspondentes.
Chama os métodos apropriados para calcular e exibir a área e o perímetro.
