Descrição do Projeto
Este projeto consiste em uma calculadora de formas geométricas implementada utilizando o paradigma de orientação a objetos. Ele permite calcular a área e o perímetro de diferentes formas geométricas, incluindo círculos, quadrados e triângulos. Cada forma geométrica é representada por uma classe separada, e os cálculos são realizados através de métodos dessas classes.

Como Usar o Programa
Execução do Programa:

Certifique-se de ter um ambiente de execução adequado para a linguagem em que o código foi implementado.
Execute o programa.
Escolhendo a Forma Geométrica:

O programa solicitará que você escolha uma forma geométrica: círculo, quadrado ou triângulo.
Fornecendo as Dimensões:

Dependendo da forma escolhida, o programa solicitará as dimensões necessárias:
Para o círculo, você precisará fornecer o raio.
Para o quadrado, você precisará fornecer o lado.
Para o triângulo, você precisará fornecer a base e a altura.
Visualizando os Resultados:

O programa calculará a área e o perímetro com base nas dimensões fornecidas.
Os resultados serão exibidos no console.
Estrutura do Código
O código é organizado em três classes:

Classe Circulo:

Atributos: raio
Métodos: CalcularArea(), CalcularPerimetro()
Classe Quadrado:

Atributos: lado
Métodos: CalcularArea(), CalcularPerimetro()
Classe Triangulo:

Atributos: base, altura
Métodos: CalcularArea(), CalcularPerimetro()
Função Principal:

Solicita a forma geométrica desejada e suas dimensões.
Cria objetos das classes correspondentes.
Chama os métodos apropriados para calcular e exibir a área e o perímetro.
